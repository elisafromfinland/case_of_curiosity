# The Case of Curiosity

### Research project 
This code was written for my MSc thesis, which examined whether frontal theta oscillations during spatial exploration could be predicted using a computational model of curiosity. In this project, 36 participants completed a virtual navigation task across novel and familiar environments while scalp EEG was recorded. Dynamic frontal theta power was predicted from time-aligned curiosity estimates generated by a self-supervised model of intrinsic motivation (Random Network Distillation; Burda et al., 2018). 

### What does the code do? 
This code implements the main computational modelling pipeline. It:
1. preprocesses images (RGB conversion, normalisation, PyTorch tensor formatting)
2. defines and trains an RND neural network (consisting of target and predictor networks) on participant navigation screenshots (1 frame/second) and generates frame-wise curiosity scores for each participant
3. aligns curiosity scores with dynamic EEG theta power
4. predicts theta power from curiosity scores using ridge regression
5. visualises condition-level R² values across participants
6. evaluates prediction performance using permutation testing

The model is trained separately for each participant while keeping the original order of exploration sequence.
